Getting started with your new BeagleBone

This is the quick-start guide. For more detailed background, see the full presentation available at: http://beagleboard.org/beagleboard101

Step #1: Install the USB-to-serial device driver

Your BeagleBone comes with a built-in USB-to-serial interface adapter. To enable it, you'll need to install the Windows, Mac OSX or Linux driver.

  • Windows: If you've installed Code Composer Studio, you've already installed the driver and you can skip this step. Otherwise, open the "\Drivers\Windows\FTDI_Driver" or "\Drivers\Windows\FTDI_64bit_Driver" folder
  • Mac OSX: Open the "/Drivers/MacOSX/FTDI_Driver" folder

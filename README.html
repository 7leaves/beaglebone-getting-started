Getting started with your new BeagleBone

This is the quick-start guide. For a more detailed background and to get the latest demonstration image, see the full presentation available at: http://beagleboard.org/beagleboard101. Until the files are copied to an external server, you can find some code and documents on the TI Internal BeagleBone wiki page.

Step #1: Install the USB-to-serial device driver

Your BeagleBone comes with a built-in USB-to-serial interface adapter. To enable it, you'll need to install the Windows, Mac OSX or Linux driver.

  • Windows:
    • If you've installed Code Composer Studio v5, you've already installed the driver and you can skip this step. Code Composer Studio v5 can be used with the BeagleBone for free. See the Code Composer Studio v5 wiki page for download and installation information.
    • Otherwise, point your Windows machine to the "\Drivers\Windows\FTDI_Driver" folder for the drivers. If you closed the window, double-click each of the .inf files to manually install the drivers.
    • Start Hyperterminal and point it to the newest COM port.
  • Mac OS X:
    • Open the "/Drivers/MacOSX/FTDI_Driver" folder, double-click the .dmg file, double-click the appropriate .mpkg file and follow the instructions.
    • Add the BeagleBone to the /System/Library/Extensions/FTDIUSBSerialDriver.kext/Contents/Info.plist file with the idVendor of 1027 and idProduct of 42704.*
    • screen /dev/tty.usb* 115200
  • Linux: 
    sudo modprobe ftdi_sio vendor=0x0403 product=0xa6d0
screen /dev/`dmesg  | grep FTDI | grep "now attached to" | tail -n 1 | awk '{ print $NF }'` 115200

Step #2: Install the USB-to-Ethernet device driver

The Linux distribution running on your BeagleBone supports emulating a USB-to-Ethernet device such that you can access your BeagleBone with networking protocols, such as web browsing.

  1. Eject the BEAGLE_BONE USB drive.
  2. Login using 'root' over your serial interface to the BeagleBone.
  3. Issue the following commands to your BeagleBone at the root@beaglebone:~# prompt: 
    systemctl stop storage-gadget-init.service
rmmod g_mass_storage
modprobe g_ether
ifconfig usb0 10.0.0.1 netmask 255.255.255.252
  4. Do the driver installation
    • Windows:
      • Point your Windows machine to the "\Drivers\Windows\RNDIS" folder for the driver. If you closed the window, double-click the linux.inf file to manually install the driver.
      • Find the new Ethernet adapter and give it an IP address of 10.0.0.1 and a netmask of 255.255.255.0.
    • Mac OS X:
      • Open the "/Drivers/MacOSX/USBCDC-G_ether" folder, double-click the .pkg file and follow the instructions. *
    • Linux: 
      ifconfig usb0 10.0.0.2 netmask 255.255.255.252

Step #3: Navigate to your BeagleBone's web interface

The Linux distribution running on your BeagleBone is configured to start a handful of web servers upon boot. This is the easiest way to get explore and program your BeagleBone if you are not already experienced with Linux.

  • For Matrix, visit http://10.0.0.1:8080/index.html.
  • For Cloud9 IDE, visit http://10.0.0.1:3000.
